# claude-skills
Collection of useful and semi-useful skills for Claude

## Looking Glass

After downloading `looking-glass.zip`, add your `.env` to the package (see `.env.example` inside the zip for the template). You will need a Vertex AI API key created in the Google Cloud Console.

```sh
zip looking-glass.zip .env
```

## Virtual Try-On

After downloading `virtual-try-on.zip`, add your `.env` to the package (see `.env.example` inside the zip for the template). You will need an OpenAI API key created at https://platform.openai.com/api-keys.

```sh
zip virtual-try-on.zip .env
```
