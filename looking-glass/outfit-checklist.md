# Outfit Analysis Checklist

This checklist is your internal analysis tool. Work through it **top to bottom** for every photo. For each item, look at the photo and describe what you see with maximum precision.

**Visibility rules (apply to every item):**
- **Skip items that are not visible or not present** — never fabricate or infer hidden details.
- If something is **partially obscured**, describe only what is visible and note the occlusion.
- For each garment or accessory, note **left/right placement** when asymmetric or side-specific (e.g., which wrist, which shoulder).
- If the photo shows a **full-body garment** (dress, jumpsuit, overalls), skip the separate upper/lower body sections and go to Section 9 instead.

The goal is zero missed details. Every visible attribute of the outfit must be captured before you compose the output prompt.

---

## 1. Overall Impression

Before zooming in, capture the big picture:

- [ ] **Style archetype** — what category does this outfit fall into? (e.g., business formal, smart casual, streetwear, athleisure, bohemian, minimalist, preppy, punk, vintage, avant-garde, workwear, resort, evening/cocktail, loungewear)
- [ ] **Formality level** — on a scale from ultra-casual to black-tie, where does this land?
- [ ] **Overall silhouette** — what is the body outline created by the clothing? (e.g., fitted throughout, oversized top with slim bottom, A-line, boxy, tailored, voluminous)
- [ ] **Dominant color palette** — list the 2–4 most prominent colors across the entire outfit, using precise color names (not just "blue" — say "dusty slate blue" or "saturated cobalt")
- [ ] **Color harmony type** — monochromatic, analogous, complementary, triadic, neutral-based, or high-contrast?
- [ ] **Total visible layers** — count distinct garment layers from innermost visible to outermost
- [ ] **Visual weight distribution** — where does the outfit draw the eye? Top-heavy, bottom-heavy, balanced, or accent-driven?
- [ ] **Era or decade influence** — does the outfit reference a specific fashion era? (e.g., 70s boho, 90s grunge, Y2K, modern minimalist)
- [ ] **Coordination level** — does the outfit appear deliberately coordinated, intentionally mismatched, or casually assembled?
- [ ] **Matching set / suit** — are any pieces clearly part of a matching set? (e.g., suit jacket + trousers in same fabric, co-ord set, matching tracksuit, twin set) Note which pieces match and in what way (same fabric, same color, same pattern).
- [ ] **Prominent text and logos** — scan the entire outfit for any clearly readable text, brand names, logos, or graphic symbols. Transcribe all visible text exactly. Note placement and size.

- [ ] **Midriff / waist** — covered, partially exposed (crop top gap), fully exposed
- [ ] **Upper arms** — covered (full sleeves), partially exposed (short sleeves), fully exposed (sleeveless/strapless)

- [ ] **Ankles / feet** — covered, ankle exposed, feet exposed
- [ ] **Overall exposure level** — minimal (most skin covered), moderate, significant
